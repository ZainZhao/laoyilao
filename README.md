# 捞一捞 校园二手市场 微信小程序



## 前言

致力于打造一个不做在线交易的校园二手信息发布网站，垂直，极简。（寻找运营合作）。


本仓库包含部分微信小程序短、后端源码。

| 系统 | 描述         |
| ---------- | ---------------- |
| laoyilao-weapp |     微信小程序前端     |
| laoyilao-back  | 后端 |
